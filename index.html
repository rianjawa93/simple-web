Tentang

Simple Web adalah template halaman landing page yang dibuat dengan HTML, CSS, dan JavaScript murni. Tanpa framework, tanpa build step — cukup buka file dan langsung jalan.

🚀

Zero Dependencies

Tidak perlu npm install. Cukup clone repo dan buka di browser.

🌙

Dark Mode

Toggle tema gelap/terang yang tersimpan di browser Anda.

📦

GitHub Ready

Siap di-deploy ke GitHub Pages dengan satu klik.
